Roadmap IFP — instantané d'avril 2026

Ce qui suit reflète ce qui a été partagé lors de la livraison de l'atelier d'avril 2026. La roadmap IFP d'Anaplan évolue fréquemment — vérifiez le statut actuel sur Anapedia et sur la Anaplan Community avant de le partager avec les clients.

ℹ Actualité du contenu

Cette page est figée à la livraison de l'atelier d'avril 2026. Pour la roadmap IFP actuelle, consultez Anapedia (le hub de documentation Anaplan) et le portail Anaplan Community Partner. Les éléments de roadmap mentionnés lors d'un atelier partenaire peuvent précéder les annonces publiques.

Capacités IFP à venir

1

Application CapEx Planning

« Plus tard cette année, nous publierons une application pour les projets CapEx — si vous avez une mission client dans les 6 prochains mois, ce sera du sur-mesure pour l'instant. » Une application CapEx Planning dédiée est sur la roadmap IFP. En attendant sa disponibilité, les exigences CapEx sont traitées comme une extension personnalisée ou une construction Anaplan séparée.

2

Évolution d'ADO

L'Anaplan Data Orchestrator devrait continuer à bénéficier d'investissements dans IFP 2.0 — en particulier autour de pipelines de données pré-construits pour les sources ERP courantes et un meilleur reporting d'erreurs pour les exécutions de pipeline. La configuration ADO actuelle dans IFP nécessite une configuration manuelle des pipelines dans la plupart des environnements clients.

3

Roadmap Headcount Planning 2.0

Headcount Planning 2.0 (publié avec IFP 2.0) fait l'objet d'itérations. Les améliorations prévues incluent des types de Planning Method supplémentaires et une intégration plus étroite avec les scénarios de workforce planning. Surveillez la Anaplan Community pour les notes de version.

4

Améliorations de l'Application Framework

L'Application Framework Configurator devrait étendre sa couverture de configuration — réduisant le nombre de cas où les clients ont besoin d'extensions personnalisées pour des exigences qui pourraient être traitées via la configuration. Chaque release IFP a ajouté de nouvelles questions de configuration.
