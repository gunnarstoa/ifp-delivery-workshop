Exercice d'extension 1 : ajustements manuels

Ajoutez 2 nouveaux Line Items d'ajustement manuel au Module OpEx et modifiez la formule d'agrégation

Extensions30 minPratique guidée
🌐 Avis de traduction

Ceci est une traduction automatique de la version originale en anglais. Pour la version officielle et la plus à jour, consultez la version anglaise. Signalez tout problème de traduction à votre contact Anaplan.

Exigence métier

Besoin de 2 types supplémentaires d'ajustements manuels pour les saisies de dépenses.

Solution

  • Ajouter de nouveaux Line Items au Module concerné
  • Modifier la formule d'agrégation pour inclure les nouveaux Line Items

Étapes

1

Ouvrez le Module Expense Planning

Ouvrez le Module INP OPX Expense Planning dans le modèle « Financial Planning ».

2

Ajoutez 2 nouveaux Line Items

Ajoutez 2 nouveaux Line Items dans la section « Direct Input Method » du blueprint :

  • Manual Adjustment #1
  • Manual Adjustment #2
  • Définissez le Format des nouveaux Line Items (identique à « Amount »)
  • Définissez le Write Access Driver pour chaque nouveau Line Item
  • Documentez à quoi servent les Line Items dans le champ « Notes »
3

Modifiez la formule d'agrégation

Nous devons modifier la formule du Line Item Direct Input Fcst pour inclure les 2 nouveaux Line Items :

  • Modifier la formule
  • Documenter ce que vous avez fait :
    • Utiliser le champ « Notes » du blueprint du Module
    • Tenir une liste maître des overrides dans un document hors-ligne, incluant la syntaxe originale et mise à jour de la formule
4

Mettez à jour le Line Item Subset et les paramètres Admin

Ajoutez les 2 nouveaux Line Items à la LISS (Line-Item Subset pour Show/Hide) :

  • Ouvrez le Line Item Subset « LISS: OpEx Filter »
  • Cochez le booléen pour les 2 nouveaux ajustements
  • Ouvrez le Module « ADMIN LISS OpEx Filter » et réglez la « Forecast Method » sur « Direct Input » pour chacun des 2 nouveaux Line Items
5

Vérifiez dans l'UX App

Ouvrez la page « OpEx – planning » dans l'UX App :

  • Sélectionnez le compte « 6005 Office Equipment »
  • Vérifiez que la logique fonctionne correctement
