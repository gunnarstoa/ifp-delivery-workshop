Obrigado

Você chegou ao fim do IFP Delivery Workshop. Obrigado pelo tempo, atenção e pelas perguntas que você trouxe.

Um último favor antes de fazer logout: por favor, ajude-nos a resetar o tenant de treinamento para o próximo grupo.

Checklist de limpeza do tenant

O tenant de treinamento é compartilhado por muitos grupos. Executar estas cinco etapas antes de sair mantém o tenant em estado limpo para o próximo grupo de participantes — e mantém estes workshops sustentáveis de operar.

⚠ Não exclua modelos ou apps TRAIN

Os modelos e aplicações TRAIN pré-populados (aqueles com o prefixo TRAIN, por exemplo IFPv2.0.0-train-base-financial-planning) pertencem a todos os participantes e precisam permanecer no lugar. Exclua apenas os apps, modelos e objetos que você criou durante os exercícios.

  • Exclua as aplicações que você configurou durante os exercícios — aquelas sem o prefixo TRAIN
  • Exclua os modelos gerados a partir das suas aplicações configuradas — aqueles sem o prefixo TRAIN
  • Exclua quaisquer objetos que você adicionou ao seu Dataspace atribuído durante os exercícios
  • Resete as hierarquias e perguntas de configuração para o padrão no Application Framework Configurator (uma hierarquia por vez — não há reset global)
  • Restaure o modelo Financial Planning para seu ID pré-extensão
    • Para Application Training tenants 00 a 60, o ID pré-extensão é 2064565
    • Para tenants usados em workshops EMEA, o ID pode ser diferente — confirme com o tenant admin EMEA antes de restaurar

Mais uma coisa

Se ainda não fez, por favor reserve 5 minutos para preencher a pesquisa pós-workshop. Usamos as respostas diretamente para cortar o que não agregou valor e reforçar o que agregou — seu feedback é a forma mais direta de moldar o que o próximo grupo verá.
