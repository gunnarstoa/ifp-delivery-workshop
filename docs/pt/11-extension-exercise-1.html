Exercício de extensão 1: Ajustes manuais

Adicionar 2 novos Line Items de ajuste manual ao Módulo de OpEx e modificar a fórmula de agregação

Extensões30 minPrática guiada

Requisito de negócio

Necessita de 2 tipos adicionais de ajustes manuais para inputs de despesa.

Solução

  • Adicionar novos Line Items ao Módulo afetado
  • Modificar a fórmula de agregação para incluir os novos Line Items

Etapas

1

Abra o Módulo de Expense Planning

Abra o Módulo INP OPX Expense Planning no modelo "Financial Planning".

2

Adicione 2 novos Line Items

Adicione 2 novos Line Items na seção "Direct Input Method" do blueprint:

  • Manual Adjustment #1
  • Manual Adjustment #2
  • Defina o Format dos novos Line Items (igual a "Amount")
  • Defina o Write Access Driver para cada novo Line Item
  • Documente para que servem os Line Items no campo "Notes"
3

Modifique a fórmula de agregação

Precisamos modificar a fórmula do Line Item Direct Input Fcst para incluir os 2 novos Line Items:

  • Modifique a fórmula
  • Documente o que você fez:
    • Use o campo "Notes" do blueprint do Módulo
    • Mantenha uma lista mestre de overrides em um documento offline, incluindo a sintaxe da fórmula original e atualizada
4

Atualize o Line-Item Subset e configurações de Admin

Adicione os 2 novos Line Items ao LISS (Line-Item Subset para Show/Hide):

  • Abra o Line-Item Subset "LISS: OpEx Filter"
  • Marque o booleano para os 2 novos ajustes
  • Abra o Módulo "ADMIN LISS OpEx Filter" e defina o "Forecast Method" como "Direct Input" para cada um dos 2 novos Line Items
5

Verifique no UX App

Abra a página "OpEx – planning" no UX App:

  • Selecione a conta "6005 Office Equipment"
  • Confira se a lógica está funcionando corretamente
