Bem-vindo

Você está dentro! O IFP Delivery Workshop é um programa virtual prático, totalmente no seu próprio ritmo, que conduz você por todo o fluxo de entrega do IFP 2.1 — configuração do Application Framework, geração do modelo, resolução de Error Logs e construção de extensões reais específicas de clientes do zero.

Se você já gerou um modelo IFP, encontrou um Error Log e ficou em dúvida se deveria escalar ou resolver por conta própria — este workshop foi feito para preencher exatamente essa lacuna.

O que você vai aprender

Você percorrerá todo o ciclo de vida de entrega do IFP:

  • Configuração do Application Framework — todas as 44 perguntas de configuração, de ponta a ponta
  • Geração do modelo e verificação pós-geração — o que a geração produz, o que verificar e por quê
  • Revisão do Error Log — ler, categorizar e resolver erros de geração sem precisar escalar
  • Extensões do zero — duas implementações de nível produção: um método de entrada direta com múltiplos ajustes e um método de taxa orientado por Headcount
  • Anaplan Way for Applications — a metodologia de entrega em seis fases que orienta cada projeto IFP

Por que isso importa

O IFP é entregue por meio do Application Framework — não é construído do zero. Isso faz com que configuração, geração e triagem do Error Log sejam o núcleo da competência de entrega, e não a modelagem personalizada. As decisões de configuração, a lista de verificação pós-geração e a disciplina de Data Tags para extensões não são opcionais — elas são o método de entrega.

Cada exercício acontece no seu próprio tenant de treinamento atribuído, com modelos IFP pré-carregados, usando cenários reais de planejamento (OpEx/Headcount, Insurance 3-Statement, Nuclear Plant OpEx).

Ao final, você será capaz de

#Resultado
1Configurar o Application Framework de ponta a ponta e gerar um modelo IFP funcional
2Ler e resolver Error Logs de geração sem escalar erros básicos de fórmula
3Construir uma extensão de entrada direta com camadas de ajuste
4Construir um método de planejamento por taxa orientado por Headcount
5Aplicar a disciplina Configurar vs. Estender vs. Modificar e marcar cada alteração com Data Tags

Para quem é este workshop

Consultores de entrega Anaplan em empresas parceiras que estão implementando o IFP para clientes — ou que esperam fazer isso nos próximos 12 meses. Este é um workshop para praticantes, não uma visão geral de produto.

Pré-requisitos:

  • Certificação Model Builder (MB), Solution Architect (SA) ou Master Anaplanner
  • Experiência prévia construindo ou configurando modelos Anaplan
  • Familiaridade com conceitos de planejamento financeiro (DRE, balanço patrimonial, headcount)

Seu tenant de treinamento para a semana

Foi atribuído a você um tenant de treinamento dedicado, pré-carregado com modelos IFP 2.1 para a semana da sua sessão. Toda a configuração e o trabalho de extensão acontecem nesse tenant de treinamento — não em produção. A maioria dos participantes completa o workshop, os exercícios e os vídeos de apoio em uma a duas horas por dia.

⚠ Confirme seu acesso antes de começar

Antes de qualquer outra coisa, vá para Acesso ao tenant e execute as três verificações: faça login, confirme o tenant correto e valide seus papéis e dataspace. Pular essa etapa é o motivo número 1 pelo qual os participantes ficam travados no meio do workshop.

Como participar: Ler · Assistir · Fazer

Este workshop é no seu próprio ritmo, então o padrão de engajamento é construído em torno do trabalho independente, em vez de uma sessão ao vivo com facilitador:

ModoO que aconteceSeu papel

Como usar este guia

Percorra as seções no seu próprio ritmo usando a barra lateral:

  • Primeiros passos: Acesso ao tenant, Visão geral do IFP, Anaplan Way for Applications
  • Configuração: Walkthrough do Application Framework Configurator, três exercícios de configuração, geração do modelo, etapas pós-geração, revisão do Error Log
  • Extensões: Visão geral das extensões e dois exercícios práticos
  • Referência: Em breve, Perguntas e respostas de sessões anteriores
