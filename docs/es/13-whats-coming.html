Capacidades próximas de IFP

1

Aplicación CapEx Planning

"Más adelante este año saldremos con una aplicación para proyectos CapEx — si tiene un compromiso con cliente en los próximos 6 meses, por ahora será a medida." Una aplicación de planeación CapEx diseñada para ese propósito está en el roadmap de IFP. Hasta que esté disponible, los requisitos de CapEx se abordan como una extensión personalizada o como una construcción Anaplan separada.

2

Evolución de ADO

Se espera que el Anaplan Data Orchestrator continúe recibiendo inversión en IFP 2.0 — específicamente alrededor de pipelines de datos prearmados para fuentes ERP comunes y mejoras en el reporte de errores en las ejecuciones de pipelines. La configuración actual de ADO en IFP requiere configuración manual de pipelines en la mayoría de los entornos de cliente.

3

Roadmap de Headcount Planning 2.0

Headcount Planning 2.0 (lanzado con IFP 2.0) se está iterando. Las mejoras planificadas incluyen tipos adicionales de métodos de planeación y una integración más estrecha con escenarios de planeación de fuerza laboral. Esté atento a la Anaplan Community para las notas de versión.

4

Mejoras del Application Framework

Se espera que el Application Framework Configurator amplíe su cobertura de configuración — reduciendo la cantidad de casos en los que los clientes necesitan extensiones personalizadas para requisitos que podrían manejarse vía configuración. Cada lanzamiento de IFP ha añadido nuevas preguntas de configuración.
