Erweiterungsübung 1: Manual Adjustments

Fügen Sie 2 neue Manual-Adjustment-Line Items zum OpEx-Modul hinzu und passen Sie die Aggregationsformel an

Erweiterungen30 Min.Angeleitete Übung
🌐 Hinweis zur Übersetzung

Dies ist eine automatische Übersetzung der englischen Originalfassung. Die maßgebliche und aktuellste Version finden Sie hier auf Englisch.

Geschäftsanforderung

Es werden 2 zusätzliche Arten von Manual Adjustments für Aufwandseingaben benötigt.

Lösung

  • Neue Line Items zum betroffenen Modul hinzufügen
  • Die Aggregationsformel so anpassen, dass sie die neuen Line Items einbezieht

Schritte

1

Das Expense Planning Modul öffnen

Öffnen Sie das Modul INP OPX Expense Planning im Modell „Financial Planning".

2

2 neue Line Items hinzufügen

Fügen Sie im Abschnitt „Direct Input Method" des Blueprints 2 neue Line Items hinzu:

  • Manual Adjustment #1
  • Manual Adjustment #2
  • Setzen Sie das Format der neuen Line Items (gleich wie „Amount")
  • Setzen Sie den Write Access Driver für jedes neue Line Item
  • Dokumentieren Sie im Feld „Notes", wofür die Line Items gedacht sind
3

Die Aggregationsformel anpassen

Wir müssen die Formel für das Line Item Direct Input Fcst anpassen, sodass die 2 neuen Line Items einbezogen werden:

  • Formel anpassen
  • Dokumentieren Sie, was Sie getan haben:
    • Verwenden Sie das Feld „Notes" des Modul-Blueprints
    • Pflegen Sie eine Master-Liste aller Überschreibungen in einem Offline-Dokument, einschließlich der ursprünglichen und der aktualisierten Formel
4

Line Item Subset und Admin-Einstellungen aktualisieren

Fügen Sie die 2 neuen Line Items zum LISS hinzu (Line Item Subset für Show/Hide):

  • Öffnen Sie das Line Item Subset „LISS: OpEx Filter"
  • Setzen Sie die Boolean-Markierung für die 2 neuen Adjustments
  • Öffnen Sie das Modul „ADMIN LISS OpEx Filter" und setzen Sie für jedes der 2 neuen Line Items die „Forecast Method" auf „Direct Input"
5

In der UX-App verifizieren

Öffnen Sie die Seite „OpEx – planning" in der UX-App:

  • Wählen Sie das Konto „6005 Office Equipment"
  • Prüfen Sie, dass die Logik korrekt funktioniert
